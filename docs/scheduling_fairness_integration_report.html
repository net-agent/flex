Window Jain Index

Jain 公平指数 (Jain's Fairness Index): 衡量资源分配公平性的经典指标。 公式: J = (∑xi)² / (n · ∑xi²)，其中 xi 是窗口内第 i 条流的包数，n 是流总数。 取值范围 [1/n, 1]，值为 1 表示完全公平（所有流获得相同份额），值为 1/n 表示极度不公平（所有包都属于同一条流）。 每行为一个窗口（大小 = 192 包），Bar 长度按百分比可视化。
